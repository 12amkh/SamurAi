# SamurAi

SamurAi is a simple AI-powered text summarization web application. It allows users to input long text and instantly receive a condensed summary generated by an advanced AI model.

## Features
- **Instant Summarization:** Powered by the Hugging Face `facebook/bart-large-cnn` model for fast and accurate summarization.
- **Clean UI:** Built with HTML and vanilla CSS for a minimal, premium frontend experience.
- **Backend API:** Built using Python and Flask.

## Prerequisites
- Python 3.7 or higher
- A Hugging Face API key

## Setup Instructions

1. **Clone or Download the Project**
   Make sure you are in the project's root directory:
   ```bash
   cd SamurAi
   ```

2. **Create a Virtual Environment (Optional but Recommended)**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install Dependencies**
   Install the required Python packages:
   ```bash
   pip install -r requirements.txt
   ```

4. **Environment Variables**
   Create a `.env` file in the root directory and add your Hugging Face API key:
   ```env
   HUGGINGFACE_API_KEY=your_api_key_here
   ```

5. **Run the Application**
   Start the Flask server:
   ```bash
   python app.py
   ```
   The application will be accessible at `http://127.0.0.0:5000` (or `http://localhost:5000`).

## Tech Stack
- **Frontend:** HTML, Vanilla CSS
- **Backend:** Python, Flask
- **AI Integration:** Hugging Face Inference API (`requests` library)
